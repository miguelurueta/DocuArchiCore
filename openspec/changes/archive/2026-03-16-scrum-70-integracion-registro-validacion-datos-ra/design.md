## Problem Statement

PROMPT ARQUITECTÓNICO — Integración de funciones workflow en RegistrarRadicacionEntranteAsync Funciones involucradas: AsignaDatosRegistroRadicadoWorkflow ValidaDatosRadicacionTareaWorkflowService SolicitaEstructuraRutaWorkflow Rol esperado: Arquitecto de software senior y desarrollador backend .NET OBJETIVO Implementar e integrar dentro del flujo del servicio RegistrarRadicacionEntranteAsync las funciones: SolicitaEstructuraRutaWorkflow AsignaDatosRegistroRadicadoWorkflow ValidaDatosRadicacionTareaWorkflowService Estas funciones deben ejecutarse antes del registro definitivo del radicado, con el fin de evitar registrar radicados cuando la información requerida para la tarea workflow no ha sido construida o validada correctamente. CONDICIÓN FUNCIONAL El flujo workflow solo debe ejecutarse cuando: util_tipo_modulo_envio == 2 Si esta condición no se cumple, el flujo actual de RegistrarRadicacionEntranteAsync no debe alterarse.
